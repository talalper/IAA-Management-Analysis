# Organizational Analysis: Israel Airports Authority (IAA)
Project Overview
This project provides a comprehensive management analysis of the Security Division at the Israel Airports Authority (IAA).
It was conducted as part of the "Introduction to Management" course, focusing on real-world organizational challenges and management strategies.

Key Models Applied
The analysis uses several academic frameworks to evaluate the organization: 
* Robert Katz’s Model: Analyzing management skills across different levels of the security division.
* SWOT Analysis: Identifying internal strengths and weaknesses vs. external opportunities and threats.
* Organizational Structure: Evaluating authority allocation and suggesting alternative structures.
* Leadership Theories: Analyzing leadership through the lens of Emotional Intelligence and Trait Theory.
* Control Systems: Examining the three stages and types of control within the division.

Summary of Insights
The project explores the unique organizational culture of the IAA Security Division—balancing a "Market Culture" of high targets and results with the necessity of strict security protocols. 
It includes practical recommendations for improving planning processes and leadership efficiency.
